# **App Name**: OnePageFolio

## Core Features:

- Fixed Navbar with Smooth Scroll: A fixed navigation bar at the top of the page with links to different sections that smoothly scroll the user to the corresponding section when clicked.
- Hero Section: An introductory section with an image or background, a title, a brief introduction, and a button that scrolls to the projects section.
- Blog Section: Displays a grid of example blog posts with a title, brief description, and a link to read more. Each post is presented in a modern card with hover effects.
- Projects Section: A gallery of showcased projects, each with an image, name, description, and links to a live demo or GitHub repository.
- Contact Form: A contact form allowing users to send messages, along with links to social media profiles.
- Footer: A simple footer with copyright information and quick links.

## Style Guidelines:

- Primary color: Dark blue (#1E3A8A) to convey professionalism and trustworthiness, complementing the personal blog/portfolio nature.
- Background color: Very light gray (#F9FAFB) for a clean and modern look, providing contrast and readability.
- Accent color: Muted gold (#A8926A) as an analogous color to provide highlights and calls to action.
- Headline font: 'Belleza' sans-serif for headlines, providing an elegant, fashionable feel. Body font: 'Alegreya' serif, for body text.
- Modern, minimalist layout with good spacing for readability and visual appeal.
- Subtle hover effects on blog and project cards to enhance user interaction.
